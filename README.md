支持多关键字搜索，

支持的文件格式：

• 文本文件 (.txt)

• Python 源码 (.py)

• JavaScript 源码 (.js)

• HTML 文件 (.html)

• CSS 样式表 (.css)

• XML 文件 (.xml)

• JSON 文件 (.json)

• Markdown 文件 (.md)

不支持的文件格式：

• Word 文档 (.doc, .docx)

• PDF 文件 (.pdf)

• Excel 文件 (.xls, .xlsx)

• 二进制文件
